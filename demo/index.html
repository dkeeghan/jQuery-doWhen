jQuery-doWhen Demo

What is it?

doWhen is a conditional checker that will do a number of tasks (like show, hide, click, disable) based on the values of forms that appear on the page.

Usage

All the logic appears on the element where the logic will be applied to. For example show and hide expect any kind of element (like a DIV or SPAN), click expects a clickable element (like A, INPUT or BUTTON, however could be anything), and disable expects a form element that can be disabled (like a SELECT, OPTION or INPUT).

There are two key attributes required:

  • data-do-when: A JSON string containing the ID/name attributes of form elements on the page and an array of the values that are required to run the do condition.
  • data-do-action: The type. Valid options: show, hide, disable
