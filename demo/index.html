jQuery-doWhen Demo

jQuery-doWhen is a jQuery plugin to handle customisable actions (like show, hide, disable, click) based on changing form field values.

Documentation for options and usage can be found on the jQuery-doWhen GitHub Page

Examples

One-to-one mappings

Simple show

Content A will appear when Option A is selected, Content B will appear when Option B is selected.

<label for="sel-1">Selectbox 1</label>
<select id="sel-1">
	<option value="" selected="selected">Select One</option>
	<option value="a">Option A</option>
	<option value="b">Option B</option>
</select>

<div data-do-when='{ "sel-1": ["a"] }' data-do-action="show">
	Content A
</div>

<div data-do-when='{ "sel-1": ["b"] }' data-do-action="show">
	Content B
</div>
